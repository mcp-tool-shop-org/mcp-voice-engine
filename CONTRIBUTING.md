# Contributing to MCP Voice Engine

Thank you for your interest in contributing to MCP Voice Engine! We welcome pull requests, bug reports, and feature requests.

## How to Contribute

1.  **Fork the repository**.
2.  **Create a new branch** for your feature or bug fix.
3.  **Make your changes** and commit them with descriptive messages.
4.  **Run tests** to ensure your changes don't break existing functionality.
5.  **Submit a pull request** to the `main` branch.

## Pull Requests

- Please include a clear description of the changes.
- Link to any relevant issues.
- Ensure all tests pass.

## Code Style

- We follow standard JavaScript/TypeScript coding conventions.
- Please use the provided `.editorconfig` settings.

## Reporting Issues

- Use the GitHub Issues tracker to report bugs.
- Provide as much detail as possible, including reproduction steps.
